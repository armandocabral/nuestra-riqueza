01 — Roadmap

Las 4 fases del negocio

01
Credenciales + Marca Personal
Meses 1–12 · 2026
+
Completar FPWMP (CFI) — base técnica y regulatoria para advisory
Iniciar construcción de marca en LinkedIn: 3 posts/semana sobre finanzas, inversión, ETFs, mercados para latinos
Abrir Advisor Account en IBKR Pro — familiarizarse con la plataforma de gestión de cuentas de clientes
Definir nicho inicial: profesionales y empresarios dominicanos en RD y diáspora (NY, Miami, Madrid)
Crear propuesta de valor clara: "Gestión patrimonial global desde RD, sin comisiones ocultas, con estrategia institucional"
Abrir estructura legal básica: SRL en RD + evaluar LLC en Wyoming (bajo costo, máxima flexibilidad)
02
Primeros Clientes + AUM Base
Meses 6–24 · 2026–2027
+
Captar primeros 10–15 clientes vía red cercana: familia, amigos empresarios, colegas del brokerage
Ticket mínimo inicial: $25,000 USD por cuenta — foco en clientes con capacidad real
Estrategia de inversión: portfolios ETF diversificados por perfil de riesgo (conservador / moderado / agresivo)
Completar CMSA — refuerza credibilidad en capital markets para clientes más sofisticados
Lanzar newsletter mensual para clientes actuales y prospectos — commentary de mercados + educación financiera
Meta AUM final fase: $2–5MM → Revenue fee: $20K–$50K/año
03
Escala + Diáspora + Producto
Meses 18–48 · 2027–2029
+
Expandir a diáspora dominicana — NY, Miami, Madrid: mercado masivo y desatendido con alta capacidad de ahorro
Registrar RIA en EE.UU. (Wyoming o Florida) para acceder formalmente a clientes americanos y dominicoamericanos
Lanzar primera herramienta propia: dashboard de portfolio para clientes + calculadora de metas financieras
Incorporar advisory de planeación financiera personal: retiro, educación universitaria hijos, protección patrimonial
Explorar partnerships con contadores, abogados, y asesores de negocios en RD como canal de referidos
Meta AUM final fase: $20–35MM → Revenue fee: $200K–$350K/año
04
Firma + SaaS + $100MM AUM
Años 4–6 · 2029–2031
+
Formalizar como firma: contratar 1–2 asesores asociados para delegar gestión operativa de cuentas
Lanzar herramienta SaaS propia para advisors en LATAM: plataforma de gestión de portfolios + CRM de clientes
Revenue mix objetivo: 60% AUM fees + 25% SaaS/herramientas + 15% advisory premium / planeación
Explorar family office services para clientes HNW ($1MM+): estrategias más complejas, real assets, private credit
Meta AUM final fase: $75–100MM → Revenue total: $800K–$1.2MM/año

06 — Oportunidades de Producto

Herramientas propias & SaaS potencial

Prioridad Alta · Año 1–2
Client Portfolio Dashboard
Portal web para tus propios clientes: ven su portfolio en tiempo real, performance, allocación, metas de retiro. Lo construyes para ti y con el tiempo lo conviertes en producto para otros advisors LATAM.
React + IBKR API Uso interno primero SaaS potencial: $99–299/mes por advisor
Prioridad Alta · Año 1
Calculadora de Metas Financieras
Tool interactiva para prospectos: "¿Cuánto necesito invertir para retirarme con $X/mes en Y años?" Lead magnet poderoso que pre-califica clientes y demuestra expertise. Construible en días.
React / No-code Lead gen Embeddable en web
Prioridad Media · Año 2–3
Advisory CRM para LATAM Advisors
CRM especializado para advisors financieros independientes en LATAM — gap enorme en el mercado. Gestión de clientes, IPS digital, reportes de performance, billing. Tu experiencia operativa es la ventaja.
SaaS B2B $199–499/mes Mercado: 5,000+ advisors LATAM
Prioridad Media · Año 2–3
ETF Screener para Mercados Emergentes
Herramienta de análisis y comparación de ETFs con enfoque en LATAM y emerging markets. Filtros por exposición geográfica, TER, dividendos, volatilidad. Monetizable via suscripción o freemium.
Data API + React $19–49/mes B2C + B2B
Largo Plazo · Año 3–5
Plataforma de Inversión para Diáspora
Plataforma tipo "robo-advisor lite" para la diáspora dominicana y caribeña: onboarding simple, portfolios modelo, educación integrada, conexión con IBKR. El mayor upside de todos — mercado masivo y desatendido.
Fintech / PaaS Fee sobre AUM Requiere regulación + capital
Largo Plazo · Año 4–6
Financial Planning API para Fintechs
API que otras fintechs en LATAM pueden consumir para ofrecer proyecciones de retiro, análisis de riesgo, y recomendaciones de portfolios a sus usuarios. Modelo PaaS puro. Alto valor, bajo volumen de clientes.
PaaS / API Revenue share o $2K–10K/mes Requiere partnerships
Nota estratégica

Las primeras dos herramientas (Dashboard + Calculadora) las construyes para ti mismo mientras operas el negocio de advisory. El costo es mínimo — un desarrollador freelance o incluso tú mismo con Claude como co-piloto. Una vez validadas con tus propios clientes, abrirlas como producto para otros advisors en LATAM es un movimiento natural y de bajo fricción. Tu ventaja competitiva es que eres practitioner primero, product builder segundo.
